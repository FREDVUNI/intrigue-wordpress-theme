== Description ==

Intrigue is a flexible WordPress theme perfect for personal blogs, business sites, and creative portfolios. Features a sleek dark/light mode toggle, full-screen search, and customizable homepage.

= Features =
* Dark/Light mode toggle with localStorage
* Full-screen search (Cmd/Ctrl+K shortcut)
* Portfolio custom post type
* Services, Contact, and Get Quote page templates
* Widget-ready footer (3 areas)
* Custom logo support
* Accent color customization
* Responsive design
* Translation ready

== Installation ==

1. Upload the theme folder to `/wp-content/themes/`
2. Activate through the 'Appearance' menu in WordPress
3. Go to Appearance → Menus to set up navigation
4. Add widgets to footer areas in Appearance → Widgets
5. Customize colors in Appearance → Customize

== Frequently Asked Questions ==

= How do I set up the homepage? =

Go to Settings → Reading and select "Your latest posts" for the dynamic front page.

= How do I change the accent color? =

Go to Appearance → Customize → Colors and use the color picker.

= Why are "Archives" showing in my footer? =

Add your own widgets in Appearance → Widgets, or the theme will show default quick links.
